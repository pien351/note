04 / 今後の技術動向・キーワード
// 業界・市場に出始めているが、まだ標準化していない技術・概念
キーワード 領域 概要 時間軸 関連言語・技術
AIエージェント開発 AI LLMが自律的にツールを使い複数ステップのタスクを実行するエージェント設計。MCP（Model Context Protocol）・ReActパターンが普及中。LangGraph/AutoGen/CrewAIが主要フレームワーク。 今〜2年 Python, LangChain, TypeScript
RAG / ベクトルDB AIData LLMに自社データを組み込む検索拡張生成。pgvector（PostgreSQL）・Qdrant・Weaviate等のベクトルDBが急増。エンジニア必須スキル化が進む。Embeddingモデルの選択が重要。 今〜3年 Python, SQL, pgvector, Qdrant
エッジコンピューティング CloudEmbed Cloudflare Workers・Vercel Edge等でコードをユーザーの近くで実行。レイテンシ削減・コスト削減。WebAssemblyがEdgeのランタイムとして標準化しつつある。 今〜3年 TypeScript, Rust, Wasm, Go
AI支援コーディング
（Vibe Coding）		 AIWeb GitHub Copilot・Cursor・Claude等でAIがコードを生成・補完。「Vibe Coding」として非エンジニアもアプリ開発する時代へ。言語知識より「設計力・プロンプト力」の重要性が増大。 今進行中 問わない（全言語）
Rust in Linux/OS Sys Linux Kernel 6.1からRust公式採用。Windows・AndroidカーネルにもRust導入。OSレベルでのC言語置き換えが本格化。組込・セキュリティ分野でRustエンジニアの希少価値が高まる見込み。 2〜5年 Rust, C
量子コンピューティング
プログラミング		 AIData Qiskit（IBM）・Cirq（Google）・Q#（Microsoft）等の量子言語・SDK。創薬・暗号・最適化への応用研究段階。商用化は5〜10年後見込みだが専門人材育成が先行。 5〜10年 Python(Qiskit), Q#, Julia
低コード/ノーコード
+ AIコード生成		 WebAuto Bubble・Webflow・Glide等のノーコードとAIコード生成（v0, Bolt.new等）の融合。プロダクト開発の民主化加速。エンジニアはより高度な設計・インフラ・AIレイヤーへ移行が進む。 今〜3年 JavaScript, TypeScript, Python
